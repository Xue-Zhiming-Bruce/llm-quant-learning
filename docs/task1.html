TASK1 · 数据获取 · 基础概念

量化交易基础与沪深股票数据获取

本任务梳理量化交易、K 线、基本面和技术面的基础概念，并用 Python 获取 688017.SH 过去一年的日线行情数据。

阅读说明 查看 CSV
学习用途

页面内容仅用于课程学习和数据分析练习，不构成任何投资建议。

示例股票 688017.SH
数据区间 2025-06-30 至 2026-06-29
数据频率 日线
输出形式 CSV + 收盘价图

任务目标

先建立概念，再完成可复用的数据获取脚本。

学习内容

  • 理解量化交易相对手工交易的优势：纪律性、数据处理能力、回测能力和系统化风控。
  • 解释 K 线、基本面、技术面等后续任务会反复使用的基础概念。
  • 掌握通过 Tushare Pro 获取沪深股票历史行情数据的基本流程。

实现内容

  • 脚本支持通过参数切换股票代码和起止日期。
  • 输出字段包括开盘价、最高价、最低价、收盘价、涨跌幅、成交量和成交额等。
  • 保存 CSV 数据，并生成收盘价走势可视化图。

实现流程

从环境配置到数据落盘，保持后续任务可复用。

  1. 配置 Token 通过环境变量或本地 .env 文件配置 Tushare Token，避免把敏感信息写入代码。
  2. 请求行情 调用 Tushare Pro 日线接口获取指定股票在目标区间内的交易数据。
  3. 清洗排序 按交易日期排序，保留后续分析需要的价格、成交量和成交额字段。
  4. 保存输出 生成 CSV 文件和收盘价走势图，作为 TASK2 和 TASK3 的输入基础。

图表与数据

收盘价图帮助快速观察价格趋势，CSV 则用于后续技术指标和策略回测。
