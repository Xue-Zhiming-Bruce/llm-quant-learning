# TASK1：量化交易基础与 Tushare 数据获取

## 1. 量化交易相较于传统手工交易的优势

量化交易是把交易思想、数据分析、风险控制和下单规则写成可以重复执行的程序。相较于传统手工操作交易，它主要有以下优势：

1. **纪律性更强**：策略按照事先设定好的规则执行，可以减少临场情绪、贪婪、恐惧等因素对交易决策的影响。
2. **处理数据能力更强**：程序可以同时处理大量股票、历史行情、财务指标和技术指标，而人工很难长期稳定地跟踪这么多信息。
3. **回测更方便**：量化策略可以先在历史数据上测试，观察收益、回撤、胜率、波动率等指标，再决定是否实盘使用。
4. **执行效率更高**：程序可以快速识别信号并执行交易，尤其适合需要频繁监控市场或多标的交易的场景。
5. **风险控制更系统**：可以提前写好止损、止盈、仓位控制、最大回撤限制等规则，避免完全依赖临时判断。
6. **可复盘和可优化**：每次交易都有明确的输入、规则和输出，便于后续分析策略为什么赚钱或亏钱，并持续改进。

## 2. 基本概念解释

### K 线

K 线是一种用来展示证券价格走势的图形。单根 K 线通常包含四个价格：开盘价、最高价、最低价和收盘价。日 K 线表示一个交易日内的价格变化，周 K 线表示一周内的价格变化。K 线可以帮助观察价格趋势、波动范围和多空力量变化。

### 基本面

基本面是指影响股票长期价值的公司和宏观因素，例如公司的收入、利润、现金流、资产负债情况、行业竞争格局、管理层质量、政策环境和宏观经济状况等。基本面分析更关注“这家公司值多少钱”和“未来盈利能力如何”。

### 技术面

技术面是指通过价格、成交量、K 线形态、均线、MACD、RSI 等市场交易数据来分析股票走势。技术面分析更关注“市场资金正在怎样交易”和“价格趋势可能如何变化”。

## 3. Python 获取沪深股票过去一年交易数据

本目录提供了脚本 [`fetch_stock_data.py`](fetch_stock_data.py)，可以通过 Tushare Pro 获取某一只沪深 A 股过去一年内每个交易日的数据，并完成：

1. 获取股票日线交易数据；
2. 绘制每日收盘价曲线图；
3. 保存 CSV 文件，供后续任务使用。

默认示例股票为 `688017.SH`（绿的谐波）。可以通过参数换成其他股票，例如 `600519.SH`。

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置 Tushare Token

推荐使用环境变量，避免把 token 写死到代码中：

```bash
export TUSHARE_TOKEN="你的tushare token"
```

也可以复制 `.env.example` 为 `.env`，然后把 token 写入 `.env`。`.env` 文件不要上传到公开仓库。

### 运行示例

```bash
python fetch_stock_data.py --ts-code 688017.SH
```

指定时间区间：

```bash
python fetch_stock_data.py --ts-code 600519.SH --start-date 20250101 --end-date 20251231
```

### 输出文件

运行后会生成：

- CSV 数据：`data/<股票代码>_daily_<开始日期>_<结束日期>.csv`
- 收盘价曲线图：`figures/<股票代码>_close_<开始日期>_<结束日期>.png`

CSV 中主要字段含义：

- `ts_code`：股票代码
- `trade_date`：交易日期
- `open`：开盘价
- `high`：最高价
- `low`：最低价
- `close`：收盘价
- `pre_close`：昨收价
- `change`：涨跌额
- `pct_chg`：涨跌幅
- `vol`：成交量
- `amount`：成交额
